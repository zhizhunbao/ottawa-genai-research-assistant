**更新时间：** 2024年9月

## 🚀 **项目概要**

渥太华市经济发展团队与阿岗昆学院合作开发 **生成式AI研究助手**。该工具使市政工作人员能够使用自然语言查询已上传的PDF报告和ottawa.ca网站内容，生成结构化分析报告和可视化图表。

**当前状态：** 第一阶段原型 ✅ **已完成** - 准备进行利益相关者演示和第二阶段开发。

## 📱 **用户体验流程**

### **当前原型体验**
1. **首页** → 功能概览和导航
2. **聊天** → 自然语言问答模拟
3. **上传** → 文档管理界面
4. **报告** → 分析和可视化预览
5. **设置** → 个性化和无障碍选项

### **未来生产体验**
1. **登录** → 安全认证
2. **上传** → 真实PDF处理和存储
3. **查询** → AI驱动的自然语言响应
4. **分析** → 自动化报告生成
5. **导出** → PDF/Word报告下载

## 🎨 **技术架构**

### **前端技术栈**
- **React + TypeScript** - 现代组件架构
- **Tailwind CSS** - 响应式、无障碍样式
- **i18next** - 国际化框架
- **Chart.js** - 数据可视化库

### **后端技术栈（计划）**
- **Python/FastAPI** - AI服务端点
- **PostgreSQL** - 用户和文档存储
- **LangChain** - AI编排框架
- **JWT** - 认证和授权
