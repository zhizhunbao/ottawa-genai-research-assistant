## 1. 产品愿景与目标 (Vision & Goals)

### 产品概述

渥太华经济发展团队正与亚岗昆学院合作开发一款 **生成式 AI 研究助手**。该助手旨在回答自然语言问题，生成摘要，并根据 ottawa.ca 托管的经济发展更新 PDF（Q1 2022 – Q4 2025）生成可视化图表。

### 核心目标

- 为经济发展分析师构建基于 RAG 的助手。
- 实现无需系统重新部署的自然语言问答。
- 生成叙述性更新、图表和来自 PDF 语料库的演讲稿。
- 提供带有来源引用和置信度指标的可信输出。

### 成功指标 (KPIs)

| 指标                          | 目标     | 测量方法               |
| :---------------------------- | :------- | :--------------------- |
| 答案准确率 (Accuracy)         | ≥75%     | SME 根据答案键进行评估 |
| 忠实度 (Faithfulness)         | ≥90%     | 引用验证               |
| 上下文召回率 (Context Recall) | ≥85%     | 基础覆盖率分析         |
| 响应时间 (P95)                | <3s      | API 监控               |
| 用户满意度                    | >4.0/5.0 | 试点后调查             |
