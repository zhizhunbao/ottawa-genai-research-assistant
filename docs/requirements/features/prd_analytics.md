# PRD: 分析与可视化模块 (Analytics)

**状态**: 进行中
**主要 Skill**: `dev-frontend_patterns`, `dev-libs_compatibility`

---

## 1. 业务目标

为分析师提供一种将枯燥的数据点快速转化为专业直观图表的方式，减少手动制图的时间。

## 2. 功能详情

### 2.1 自动图表识别

- **逻辑**: AI 识别查询答案中的数值和维度，并推荐最适合的图表类型（Bar, Line, Pie）。
- **组件**: 使用 `recharts` 库。

### 2.2 演讲笔记 (Speaking Notes)

- **动作**: 用户点击“生成演讲稿”按钮。
- **输出**: 提取核心统计数据，生成具有叙述性（Narrative）的 bullet points。

## 3. 验收标准 (Acceptance Criteria)

### AC-1: 图表美学 (Premium Design)

- [ ] 图表背景应具有微妙的渐变或透明度。
- [ ] 悬停（Hover）在数据点上时，应显示具备毛玻璃效果的 Tooltip。

### AC-2: 功能交互

- [ ] 用户应能一键切换图表类型。
- [ ] 图表支持导出为 PNG 或 SVG 格式，且保持设计质感。
