# PRD: Phase 1 - 内部 PDF 检索与基础 RAG

**状态**: 已完成 (Maintained)
**目标**: 实现对渥太华经济发展局 PDF 报告的语义搜索、问答及引用展示。

---

## 1. 核心功能 (Core Features)

### 1.1 文档管理 (Document Management)

- **PDF 上传**: 支持管理员通过 Web 界面上传 PDF（上限 50MB）。
- **自动化解析**: 提取文本、表格，并按年度、季度打标签。
- **向量化索引**: 使用 Azure OpenAI (ADA-002) 生成嵌入并存入 Azure AI Search。

### 1.2 智能问答 (Chat & QA)

- **自然语言查询**: 支持分析师直接提问。
- **混合检索**: 结合关键词和向量相似度搜索。
- **答案生成**: 基于 GPT-4o 生成答案，必须包含引用标识。

### 1.3 来源引用 (Citations)

- **实时引用**: 答案中每个关键点必须对应 PDF 的特定页码和片段。
- **预览功能**: 点击引用可跳转或显示 PDF 片段预览。

---

## 2. 验收标准 (Acceptance Criteria)

### AC-1: 文档处理

- [ ] 系统应在 10 分钟内完成 20 页 PDF 的解析与索引。
- [ ] 必须准确识别报告的年份（2022-2025）和季度（Q1-Q4）。

### AC-2: 问答质量

- [ ] 检索到的内容与查询的相关度分数应 > 0.7。
- [ ]生成的回复中，引用来源的准确率必须达到 90%。

### AC-3: UI 表现

- [ ] 聊天列表页应采用流式布局。
- [ ] 加载状态应显示骨架屏以增强 Premium 体验感。
